# Stray Dog Hider

A RuneLite plugin that hides annoying stray dogs that follow you around.

## Features
- **Varrock Stray Dog:** Hides the standard stray dog in Varrock.
- **Clan Hall Dog:** Hides the stray dog found in the Clan Hall.
- **Varlamore Molossus:** Hides the Molossus dogs in Varlamore.

## Configuration
You can toggle each dog individually in the plugin settings panel.
